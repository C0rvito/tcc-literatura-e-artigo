# Overview do TCC

Este documento apresenta uma síntese geral da pesquisa sobre a **Modelagem de Redes Biológicas para a Coexpreensão da Convergência Fenotípica na Doença de Parkinson**.

O objetivo é integrar dados de múltiplas fontes para identificar como diferentes vias biológicas convergem para o fenótipo da Doença de Parkinson.
